1. Build Tool: Zodiac

Even to create a small site with 3-4 pages can get complicated. Setting up a node-based build tool for such a small site is difficult to teach to beginners as there are lots of ways for command line tools to go wrong, so I searched for a simple static site generator written in awk and sh. I found one on github here: zodiac The basic usage instructions are a bit cryptic for beginning coders, so I plan to fork this and make it more understandable.
